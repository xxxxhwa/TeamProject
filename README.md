# TeamProject 
-프로젝트 개요 
오픈cv를 최대한 활용하여 사용자의 동작을 인식하고 인식한 동작을 출력하는 프로그램을 구상해 보았습니다. 최근 모션캡처를 활용한 컨텐츠들이 빠른 속도로 늘어나고 있는 상황에서 비슷한 소프트웨어의 구현은 좋은 경험이 될 것이라 판단합니다.
